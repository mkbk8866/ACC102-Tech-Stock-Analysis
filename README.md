# ACC102 Individual Mini Assignment
## Tech Stock Analysis | Apple · Microsoft · Google

### 1. Project Background & Analytical Problem
Many individual investors want to invest in leading technology stocks but do not know how to compare return, risk, and stability among large tech companies.
They often struggle with these key questions:
- Which company achieved higher return in 2024?
- Which stock has lower risk (volatility)?
- Which one provides better risk‑adjusted performance?

This project uses Python and real stock data to answer these questions and help investors make clear, data‑driven investment decisions.

### 2. Project Overview
This project focuses on three representative technology companies: Apple, Microsoft, and Google.
Using Python and the WRDS CRSP database, this project collects daily stock data for 2024, conducts data cleaning, financial indicator calculation, and visual analysis.

### 3. Data Source
- Database: WRDS (Wharton Research Data Services)
- Dataset: CRSP Daily Stock File
- Period: 2024-01-01 — 2024-12-31
- Analyzed Firms: Apple, Microsoft, Google

### 4. Analysis Content
- Normalized stock price trend comparison
- Cumulative return performance analysis
- Risk‑return characteristics and scatter plot
- Rolling 30-day volatility measurement
- Daily return correlation heatmap
- Daily return distribution visualization
- Regression analysis (Apple as the benchmark)

### 5. Key Findings
- Microsoft achieved the highest annual return in 2024.
- Google showed the lowest volatility and relatively stable performance.
- The risk‑return profile helps investors choose according to their risk preference.

### 6. File List
- `stock_analysis.ipynb`: Complete runnable Python code
- 6 visual output charts in PNG format
- `README.md`: Project introduction document
